Google Fu

Some tips and tricks to help you navigate through Google.
  • search for google syntax
  • click on link google search operators : the complete list
  • the page is an overview of Google commands you can use to help narrow your results
  • we will go through some of the examples but you are urged to go through the webpage and learn the commands.  The better your searches are the easier it will be for you to troubleshoot (debugging) and research (recon).
  • if i search for telsa in google i will get the website along with anything associated with the company (newsreports, social media accounts, fans, etc).  
  • What if i just wanted results from their official website?  We can use Google key words to filter our searches.
  • site:tesla.com will just search for pages from the tesla website
  • site:tesla.com -www will remove www from all the searches. so it will return websites like shop.tesla.com.
  • This is a good way to find subdomains also !
  • We can seach for the file type - we just add the filetype: keyword followed by th etype of file you are searching against the website you are Googling .
  • site:tesla.com filetype:pdf this is looking for pdf files ot can search for any file type (csv, xlsx, etc).
  • We do this to find some potenitally sensituve files or information that you may find as a surprise!
