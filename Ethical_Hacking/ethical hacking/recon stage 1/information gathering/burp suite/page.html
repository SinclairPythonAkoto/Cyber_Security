Burp Suite

Burp Suite is a web proxy that intercepts traffic.  It is a built in tool that you can find in Kali Lunix.

Open the app and click on Tempory Project then Next.

Now we need to set up the Firefox in Kali to be able to use the Burp Suite.

Click on the hanburger sign then Preferences.

Scroll to the bottom to find Settings in the Network Proxys.

Select Manual Proxy Configuration and make sure the HTTP Proxy is set to 127.0.0.1 and the Port is set to 8080.  Before saving and exiting, make sure you check the box Use this proxy for all protocols.

Go to https.burp, click on the CA Certificate to download and save the file.

Now go back to the Firefox settings and navigate to the Certificates in Privacy & Security.  Click on View Certificates and import the certificate you just downloaded.

Select both check boxes then the OK button.  After these steps it should be now ready to use.


Burp Suite is a good tool because it allows you to seach websites and get a lot of information from the pages etc.  This is still classed as passive reconnasissance.

The Burp Suite has a built in web browser so the steps above are no longer needed.
