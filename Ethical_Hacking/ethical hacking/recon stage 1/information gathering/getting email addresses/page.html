Discovering Email Addresses

We will look at some of the tools used to look up email addresses and find people.

Imagine we want to get in contact with someone in a company, we have looked them up in their company website but
there is no direct email address.  In this section we will look at how we would be able to retrieve an email address in this scenario.

hunter.io
You enter the name of the company to check if the format of the company's email.  With the format of the company email address, we can then use that to create our email.  To use the website we need to sign up - it's free.

In our Kali Linux go to hunter.io then sign up.


phonebook.cz
You enter the domain name then choose if its emails, domains or URLs you are searching for.  The app will return a list of results.

This is totolly free without the need to sign up.

Clearbit
Is a Google extension that can find people's email address within a company.  You have to use it with Google Chrome.

Email Hippo
Use this to verify if an email exists.  It will return an OK or Bad result.  Find the wibsite at tools.verifyemailaddress.io.

You can also use email-checker.net/validate which is a different website to Email Hippo with the same type of service.


REMEMBER !!
  • If you have a target's email, you can click forgot password.  Sites like Google will show you snippets of the email where they will send the verification code to.  With that you can try to crack the users email.
